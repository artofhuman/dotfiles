## Dotfiles

### installation

```sh
git clone https://github.com/artofhuman/dotfiles.git ~/.dotfiles && cd ~/.dotfiles && script/bootstrap
```

## Mac applications

```sh
brew bundle
```

### change to zsh

```
chsh -s $(which zsh)
```

### Install plug for NeoVim
https://github.com/junegunn/vim-plug#neovim

if need add `which zsh to /etc/shells`
